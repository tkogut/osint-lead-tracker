# PLAN-027: AI Debug Terminal & Raw Execution Tracer in Piaskownica LLM

**Status:** IN_PROGRESS  
**Date:** 2026-07-23  

---

## 🎯 Goal
Add an interactive **"📟 Podgląd Egzekucji LLM (Debug Terminal)"** below the "Wynik AI" panel in Piaskownica AI. It will display the exact execution trace of Gemini, including:
1. **Sent System Instruction & Prompt**: The full text sent to Gemini (after `format_prompt_dates` and template wrapping).
2. **Raw Model Response**: Unparsed text response generated by Gemini before JSON filtering/parsing.
3. **Execution Metrics**: Input tokens, output tokens, and execution latency (ms).

This will allow instant diagnosis of why Gemini accepts or rejects specific offer texts.

---

## 🏗️ Implementation Details
1. **Backend (`src/main.py`, `src/schemas.py`)**:
   - Update `run_sandbox_test` to measure execution time, capture `usage_metadata` (tokens), capture exact `contents_to_send` and `formatted_prompt`, and capture `raw_model_response`.
   - Include `debug_info` object in `/api/sandbox/test` JSON response.

2. **Frontend (`src/static/index.html`, `src/static/app.js`)**:
   - Add a dark code terminal component below "Wynik AI" in `index.html`.
   - Add toggle tabs in the terminal: `[Surowa Odpowiedź LLM]`, `[Wysłany Prompt Systemowy]`, `[Wysłana Treść]`, `[Metryki & Tokeny]`.
   - Bind API `debug_info` to render live in the terminal upon clicking "Uruchom Test AI".
